# bunnyc_mgrweb
bunnyc 运维管理平台

# 定位
1. js vue 的练习项目
2. 运维平台，纯前端项目，通过调用 bunnyc_mgr 项目的API开展功能
3. 定位之一是运维平台中“管”的“可视化”
4. 定位之二是运维相关的web系统入口；

# 概念
## 可视化说明
重点是体现出系统当前的状态，例如：最近一小段时间（1小时）的信息展示，或者全局状态信息；
更全面的历史状态数据，还是应该由专业的监控系统进行展示，例如grafana
